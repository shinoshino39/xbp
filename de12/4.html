ヒットアンドブロー

ヒットアンドブローとは

ヒットアンドブローとは、
相手が決めた重複しない数字や文字の並びを、
「ヒット（位置も一致）」と
「ブロー（位置は違うが含まれている）」というヒントをもとに当てるゲームである。

ソースコード

数字だとコードが複雑になってしまうので、今回は答えを文字にする。

import random

# 答えを作成（A〜Fの中から4文字を重複なしで選ぶ）
answer = random.sample(['A', 'B', 'C', 'D', 'E', 'F'], 4)

print("=== ヒット＆ブローゲーム ===")
print("A〜Fの中から4文字を予想して入力してください（例：ABCD）")

while True:
    guess = input("予想を入力: ").upper()  # 大文字に統一
    
    # 入力チェック
    if len(guess) != 4 or any(c not in 'ABCDEF' for c in guess):
        print("⚠ A〜Fの4文字で入力してください。")
        continue
    if len(set(guess)) != 4:
        print("⚠ 同じ文字を使わないでください。")
        continue

    # ヒットとブローを数える
    hit = 0
    blow = 0
    for i in range(4):
        if guess[i] == answer[i]:
            hit += 1
        elif guess[i] in answer:
            blow += 1

    print(f"{hit} ヒット, {blow} ブロー")

    # 4ヒットならゲーム終了
    if hit == 4:
        print(f" 正解！答えは {''.join(answer)} でした！")
        break

コードの説明

部分　　内容
random.sample()　　　　　　　　A〜Fの中から4文字をランダムに選ぶ
input().upper()　　　　　　　　小文字で入力されても大文字に変換
any(c not in 'ABCDEF' for c in guess)　　入力チェック（A〜F以外を防ぐ）
len(set(guess)) != 4　　　同じ文字を使ったら警告
while True:　　　　　　　　正解するまで繰り返すループ
hit == 4　　　　　　　　　すべて一致したら終了

感想

わからないコードはネットを駆使して理解しながらゲームを作れた。
更に用語を理解できればより複雑なミニゲームが作れそうだ。
